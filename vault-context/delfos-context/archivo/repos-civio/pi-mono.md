# pi-mono

Monorepo de herramientas para construir agentes de IA y gestionar despliegues de [[LLM]]. Es un outlier moderno dentro del ecosistema Civio — no está relacionado con periodismo de datos.

## README

> **Looking for the pi coding agent?** See `packages/coding-agent` for installation and usage.

### Paquetes

| Paquete | Descripción |
|---------|-------------|
| `@mariozechner/pi-ai` | API unificada multi-proveedor (OpenAI, Anthropic, Google, etc.) |
| `@mariozechner/pi-agent-core` | Runtime para agentes con tool calling y state management |
| `@mariozechner/pi-coding-agent` | CLI interactivo de agente de codificación |
| `@mariozechner/pi-mom` | Bot de Slack que delega mensajes al agente de codificación |
| `@mariozechner/pi-tui` | Librería TUI con renderizado diferencial |
| `@mariozechner/pi-web-ui` | Web components para interfaces de chat con IA |
| `@mariozechner/pi-pods` | CLI para gestionar despliegues vLLM en GPU pods |
