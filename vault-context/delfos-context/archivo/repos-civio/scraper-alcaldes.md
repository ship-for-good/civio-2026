# scraper-alcaldes

Scraper para extraer la lista de alcaldes de España desde la web del [[Ministerio de Política Territorial]].

## README

> Los ficheros están en URLs tal que así:
> `https://ssweb.seap.minhap.es/portalEELL/consulta_alcaldes/alcaldes?id_provincia=3`

## Contexto

- Scraper simple en Shell
- Recorre todas las provincias para obtener el listado completo de alcaldes
- Pipeline de datos oficiales → dataset explotable
