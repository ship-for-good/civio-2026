# scraper-elecciones-generales-2015

Scraper de resultados de las [[Elecciones Generales 2015]].

## README

```bash
bundle install
ruby fetch_congress.rb > congress_results.csv
ruby fetch_senate.rb > senate_results.csv
ruby fetch_census.rb > census.csv
```

## Contexto

- Scraper histórico
- Extrae datos de Congreso, Senado y censo
- Archivado tras las elecciones
