# civio-graphs-lib

Librería de [[Civio]] para el desarrollo de gráficos y visualizaciones. Publicada como paquete npm.

## README

- Publicada en npm como `civio-graphs-lib`
- Usada en el repositorio [[civio-graphs]]
- Ejemplos disponibles en [[graphs.civio.es]]

### Instalación

```bash
npm install civio-graphs-lib
```

## Contexto

- Librería compartida para el ecosistema de visualizaciones de Civio
- Complementa a [[civio-graphs-boilerplate]] y [[civio-graphs-public]]
