# civio-graphs-public

Repositorio moderno de visualizaciones de datos abiertas utilizadas en investigaciones periodísticas de [[Civio]].

## README

### Visualizaciones incluidas

| Gráfico | Descripción | Stack |
|---------|-------------|-------|
| [[Motivos de sanciones de aislamiento]] | Square-packing interactivo mostrando sanciones de aislamiento en prisiones por código disciplinario (2020-2025). Anillos concéntricos por motivo, filtrable por administración y género. | Svelte 5, Vite 7, D3 |
| [[Sanciones por administración penitenciaria]] | Barras apiladas comparando sanciones de aislamiento en las tres administraciones (AGE, Cataluña, País Vasco). | Svelte 5, Vite 7, D3 |
| [[Tiempos en el limbo]] | Waffle chart basado en Canvas visualizando tiempo en aislamiento en prisiones españolas. 7.800 casos distribuidos en 5 grupos de duración. | Svelte 5, Vite 7, D3 |

## Contexto para hackathon

- Stack moderno: [[Svelte 5]], [[Vite 7]], [[D3.js]]
- Repositorio activo y en evolución
- Buen ejemplo de storytelling interactivo con datos públicos
- Código fuente reusable para crear nuevas visualizaciones periodísticas
