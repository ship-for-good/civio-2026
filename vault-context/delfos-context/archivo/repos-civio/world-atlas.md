# world-atlas

Generación de archivos [[TopoJSON]] del mundo desde datos de [[Natural Earth]].

## README

> Generates TopoJSON files from Natural Earth's vector data, version 2.0.0.

### Formatos

- `world/110m.json` — escala 1:110m (small scale)
- `world/50m.json` — escala 1:50m (medium scale)

Cada archivo contiene: `countries` y `land` como geometry collections.

## Contexto

- Herramienta de datos geoespaciales
- Fork/repo del proyecto original de [[Mike Bostock]]/[[D3]]
- Precedente de [[es-atlas]]
- Complementa visualizaciones geográficas de Civio
