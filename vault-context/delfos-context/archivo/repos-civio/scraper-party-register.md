# scraper-party-register

Scraper del [[Registro de Partidos Políticos]] del [[Ministerio del Interior]].

## README

```bash
ruby fetch.rb   # Descarga todas las páginas del Registro
ruby parse.rb   # Extrae datos de cada partido → partidos.csv
```

## Contexto

- Scraper de una fuente oficial clave
- Extrae datos estructurados de partidos políticos
- 10 estrellas — popular entre los scrapers de Civio
- Archivado pero el dataset sigue siendo valioso
