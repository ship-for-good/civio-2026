# medicamentalia

Investigación periodística sobre acceso a medicamentos en el mundo.

## README

> A journalistic investigation into access to medicines around the world.

### Stack

- [[Jekyll]] - generador de sitio estático
- [[Gulp]] - task runner
- [[Bootstrap 3.3]] - framework frontend

### Requisitos

- Ruby 2+
- NodeJS y NPM

### Reproducción de contenido

Los datos y contenido periodístico están bajo licencia [[CC-BY 3.0]]. Las fotografías de terceros tienen otros términos.

## Contexto

- Proyecto de investigación internacional
- Sitio estático con Jekyll — simple pero efectivo
- Contenido reutilizable con atribución
