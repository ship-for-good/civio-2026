# gulp-boilerplate

Boilerplate [[Gulp]] con [[Browsersync]], [[Browserify]], [[Babel]] y [[Sass]].

## README

### Uso

```bash
npm install
npm start
```

### Estructura

- `src/js` — JavaScript fuente
- `src/scss` — Estilos Sass
- `dist/js` y `dist/css` — Archivos compilados

## Contexto

- Herramienta auxiliar, no relacionada con la misión principal
- Template para proyectos frontend antiguos de Civio
