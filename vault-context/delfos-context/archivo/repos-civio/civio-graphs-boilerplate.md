# civio-graphs-boilerplate

Boilerplate para desarrollo de gráficos con [[Webpack 4]], [[Babel 7]], Sass y hot reload.

## README

> Used for graphs development in [[civio-graphs]].

### Características

- ES6 con Babel 7
- CSS con Sass y Autoprefixer
- Servidor local con live reload (Webpack dev server + css-hot-loader)

### Comandos

- `npm run dev` — servidor de desarrollo en `localhost:5000`
- `npm run build` — genera `dist/` con `index.html`, `main.css` y `main.js`

## Contexto

- Herramienta auxiliar para el ecosistema [[civio-graphs]]
- Sirve como template para nuevas visualizaciones
